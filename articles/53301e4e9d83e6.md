## メモ

```sh
# キャッシュなしでビルド
docker-compose build --no-cache

# 特定のサービスのみ再ビルド、再起動
docker-compose up -d --no-deps --build [サービス名]

# 全てのイメージを削除
docker rmi $(docker images -q)
```
